Sistema de Consulta de Funcionários
Desenvolvimento Web
2025

Desenvolvi um sistema web responsivo completo para consulta integrada de dados corporativos de funcionários, implementando interface moderna com filtros avançados e validação em tempo real. O projeto envolveu integração direta com SQL Server através de API REST autenticada, utilizando consultas parametrizadas para garantir segurança e performance.
A solução consta com sistema de busca multi-critério (CPF, nome, função, cidade, segmento), interface adaptativa para desktop/tablet/mobile com design responsivo, e tratamento robusto de estados de loading e erros. Implementei técnicas de otimização como validações, máscaras automáticas para CPF e painéis expansíveis para visualização detalhada de dados, resultando em uma ferramenta intuitiva que reduziu significativamente o tempo de consultas corporativas.
Sistema criado especificamente para as equipes de Suporte N1 e N2, eliminando processos manuais de busca de dados de colaboradores e aumentando significativamente a agilidade e eficiência no atendimento de suporte técnico interno.
